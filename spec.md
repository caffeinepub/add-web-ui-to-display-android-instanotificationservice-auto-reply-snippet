# Specification

## Summary
**Goal:** Add a developer-focused web page that displays the provided Android Manifest service declaration and Kotlin `InstaNotificationService` code as syntax-highlighted, copyable snippets.

**Planned changes:**
- Create a single frontend page with two clearly labeled sections: AndroidManifest.xml snippet and Kotlin service class snippet.
- Render each snippet in a formatted, syntax-highlighted monospace code block with preserved whitespace/indentation.
- Add a one-click "Copy" button for each snippet that copies the exact snippet text to the clipboard.
- Apply a consistent developer-oriented visual theme across layout, typography, code blocks, and buttons, avoiding blue/purple as primary colors, with all UI text in English.

**User-visible outcome:** Users can view the two provided code snippets in a clean, themed UI and copy either snippet to the clipboard with one click.
